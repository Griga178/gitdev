# 📄 Word Merge Tool

Автоматический генератор документов Word на основе шаблона и данных из Excel.

## 📋 Описание

Инструмент позволяет создавать множественные копии Word-документов по шаблону, автоматически подставляя данные из Excel-таблицы. При первом запуске автоматически создает Excel-файл с необходимыми полями на основе плейсхолдеров в шаблоне.

## 📖 Инструкция по использованию

### Шаг 1: Подготовка шаблона Word

1. Создайте файл `example.docx` в папке проекта
2. Напишите текст документа
3. Добавьте плейсхолдеры в формате `{имя_параметра}`

**Пример содержимого шаблона:**
```
ДОГОВОР № {contract_number}

г. Москва                                    «{date}»

Заказчик: {client_name}
ИНН: {inn}
Адрес: {address}

Сумма договора: {amount} рублей

Подпись: _________________
```


### Шаг 2: Первый запуск (создание Excel)

```bash
python main.py
```

При первом запуске скрипт:
- ✅ Найдёт шаблон `example.docx`
- ✅ Просканирует все плейсхолдеры
- ✅ Создаст файл `param.xlsx` с заголовками


### Шаг 3: Заполнение Excel

Откройте `param.xlsx` и заполните данные:

| filename | contract_number | client_name | inn | address | amount | date |
|----------|-----------------|-------------|-----|---------|--------|------|
| Dogovor_001 | 12345 | ООО "Ромашка" | 7701234567 | г. Москва, ул. Ленина, 1 | 100000 | 15.01.2024 |
| Dogovor_002 | 67890 | ИП Иванов | 7709876543 | г. СПб, пр. Невский, 10 | 250000 | 16.01.2024 |

**Важно:**
- Первый столбец `filename` — имя будущего файла (без расширения `.docx`)
- Остальные столбцы должны точно совпадать с плейсхолдерами в шаблоне (без скобок `{}`)
- Не удаляйте первую строку с заголовками

### Шаг 4: Генерация документов

```bash
python main.py
```

### Шаг 5: Проверка результатов

Готовые файлы находятся в папке `output/`:
```
output/
├── Dogovor_001.docx
└── Dogovor_002.docx
```

## 🔧 Примеры плейсхолдеров

| В шаблоне Word | В Excel (заголовок) | Пример значения |
|----------------|---------------------|-----------------|
| `{request_number}` | request_number | 12345 |
| `{client_name}` | client_name | Иванов И.И. |
| `{date}` | date | 01.01.2024 |
| `{amount}` | amount | 100000 |


## 💡 Советы

1. **Не меняйте заголовки в Excel** — они должны точно совпадать с плейсхолдерами в шаблоне
2. **Закрывайте Excel перед запуском** — иначе может возникнуть ошибка доступа к файлу
3. **Используйте понятные имена файлов** — без специальных символов (`/`, `\`, `:`, `*`, `?`, `"`, `<`, `>`, `|`)
4. **Проверяйте шаблон** — убедитесь, что все плейсхолдеры в правильном формате `{name}`
5. **Резервная копия** — храните оригинальный шаблон отдельно от генерируемых файлов
